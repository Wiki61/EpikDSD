# EpikDSD
Helps setting up DSD and FMP



Download the DSD and FMP packages from http://www.dsdplus.com/download-2/

Place the following files in the folder called lib

FMP-Map.cfg		
lame_enc.dll
DSDPlus.bin		
FMP.EXE			
libusb-1.0.dll
DSDPlus.event		
FMP.cfg			
msvcr100.dll
DSDPlus.exe		
FMP24.cfg		
msvcrt.dll
DSDPlus.frequencies	
FMP24.exe		
pthreadVCE2.dll
DSDPlus.groups


Place the following files in the folder called docs

DSDPlus.txt	
FMPA.txt	
Trunking.txt	
Changes.txt
